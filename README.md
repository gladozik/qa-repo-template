Установка и запуск тестов:
1. Скопировать репозиторий локально
```git clone https//xxx.git```
2. Открыть терминал в корневой папке репозитория
3. Установить зависимости
```npm install```
4. Запустить тесты
```npx playwright test --workers 1```

P.S. Есть проблемы с таймаутом при регистрации от NGINX
Запускаем на 1 воркере, т.к. nginx таймаутит регистрацию
Если возникает ошибка "Sorry, the page you are looking for is currently unavailable." - нужно вручную перезапустить тест
